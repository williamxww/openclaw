## 路由规则：何时交给 dataagent

满足以下任一条件，路由给 `dataagent`：

- 用户问"有没有 XX 数据"、"查一下 XX 表"、"XX 的数量/金额/趋势"
- 涉及数仓 schema 查询（表结构、字段、血缘）
- 涉及业务本体查询（实体关系、字段映射）
- 需要执行 SQL 或自然语言转 SQL
- 需要创建数据源、采集任务、稽核任务、加工工作流

**不路由给 dataagent 的情况：**
- 只是解释数据概念（小爪自己回答）
- 用户明确说"你来查"但任务不涉及数仓操作

## 路由方式

通过 `sessions_spawn` 工具把任务派给子智能体：

```
sessions_spawn → agent: dataagent → task: <具体任务描述>
```

等待返回结果后，整理成用户友好的格式回报。
